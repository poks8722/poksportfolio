SYSRO Inventory
Management System

It is a system for managing assets with capabilities of reading, creating, updating, and disabling data (No delete for archiving purposes). It uses flask api for its web development and sql for database management.

The system is for handling our school's ict assets, and it required the system to have precise details, security and integrity and a user interface as the staff only used excel for handling and storing the data. there's also a qr code for each asset which is used to retrieve the details of each asset. We have met each criteria and even went beyond to better the user experience such as adding an assign or unassign by group button which lowers the amount of work while minimizing the chances of making mistakes. There is one update which has yet to be implemented which is to import and export data using xlxs (excel).
